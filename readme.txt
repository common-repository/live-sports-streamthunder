Get automatically a list of events of all kind of sports for your site and put your own links to drive traffic to your affiliate campaigns or landing pages.

== Description == 

With this plugin you can show automatically a daily list of events from all kind of sports on your site or just the sport you want. Put your own links on each sport event to send traffic wherever you want or show the default third party links to brind your users usefull sites to watch every sport match.

== Installation ==

1. Upload the folder 'st-widget' found inside the zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==

1. Create widgets selecting the sports for each one on Live Sports Streamhunder tab in wp-admin.
2. Copy & Paste the generated shortcode and put it on a page, post or wherever you want to show the list of sport events.
3. Configure your widget so that it best fits your site on Display Options tab in wp-admin.

= Changing the CSS =

Display Options tab in wp-admin.
